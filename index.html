2.1 Modelo de Caja (Box Model)

Todo elemento HTML es una caja compuesta por cuatro capas: content → padding → border → margin.

Diagrama del Box Model

margin
border
padding
content

Ejemplos de cajas

Caja A

box-sizing: content-box (default)

margin: 20px | padding: 20px | border: 3px solid

Caja B

box-sizing: content-box

margin: 10px | padding: 30px | border: 5px dashed

Caja C

box-sizing: border-box

padding y border incluidos dentro del width

Elementos de Bloque vs En Línea

Los elementos de bloque ocupan todo el ancho disponible:

div — elemento de bloque
otro div — empieza en nueva línea

Los elementos en línea solo ocupan el espacio de su contenido:

span 1 span 2 span 3

Con display: inline-block se combinan ambos comportamientos:

inline-block 1 inline-block 2 inline-block 3

2.2 Visualización y Posicionamiento

Propiedades CSS para controlar cómo se ubican y muestran los elementos.

A. Position

position: static — flujo normal
position: relative — desplazado 15px abajo y a la derecha
Contenedor con position: relative
position: absolute — esquina superior derecha
elemento de referencia

⚠️ position: fixed — el botón dorado fijo está anclado a la ventana del navegador (abajo a la derecha).

fixed

B. Float y Clear

float: left
float: right

Este texto fluye alrededor de los elementos flotantes ocupando el espacio disponible entre ambos.

clear: both — corta el flujo flotante

C. Display

display: block
inline inline

inline-block inline-block

display: none → OCULTO este elemento no ocupa espacio ni se renderiza.

D. Overflow

visible

Contenido que desborda sin restricción fuera del contenedor definido.

hidden

Contenido que desborda pero queda recortado fuera del contenedor.

scroll

Contenido que desborda y se puede desplazar, scroll siempre visible.

auto

Contenido que desborda y el scroll aparece solo cuando es necesario.

E. Visibility

visibility: visible — elemento visible
visibility: hidden — OCULTO pero sigue ocupando su espacio
Este elemento respeta el espacio del elemento oculto de arriba

F. Z-Index

Mayor z-index = más al frente en el apilamiento.

z-index: 1
z-index: 2
z-index: 3 (frente)
